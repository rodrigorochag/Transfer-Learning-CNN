# Transfer-Learning-CNN
Transfer Learning CNN using IMAGENET1K_V1 weights. Using Fine Tuning standard

Dataset: https://www.kaggle.com/datasets/misrakahmed/vegetable-image-dataset

21000 images from 15 classes are used where each class contains 1400 images of size 224Ã—224 and in *.jpg format. The dataset split 70% for training, 15% for validation, and 15% for testing purpose.
This dataset contains three folders:

train (15000 images)
test (3000 images)
validation (3000 images)
